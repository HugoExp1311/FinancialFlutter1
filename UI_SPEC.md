# UI/UX Specification
## Design System
- Primary Color: #6366f1 (Indigo)
- Dark Mode support: Mandatory for better UX.
- State Management: Provider/Riverpod for reactive UI updates.

## Components
- Balance Card: Displaying Net, Income, and Expense.
- Transaction Item: Icon-based category display with color coding.
- Chat Interface: Bubble-style conversation with typing indicators.
